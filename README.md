# 🛡️ EHS 맞춤 규제 진단 시스템 v4

## 핵심 컨셉
> **사장님이 MSDS PDF만 올리면 끝!**  
> CAS 번호? 몰라도 됩니다. 시스템이 알아서 합니다.

## v4 신규: 변경점관리

### MSDS(화학물질)
| 기능 | 설명 |
|------|------|
| 📥 신규 추가 | 새 MSDS PDF 업로드 → CAS 추출 → KOSHA API 조회 |
| 🔄 변경(갱신) | 기존 MSDS 업데이트 시, 이전 데이터 처리 선택 |
| 🗑️ 즉시 삭제 | 이전 데이터 바로 삭제 |
| 📅 1개월 후 삭제 | 이전 데이터 30일간 보관 후 삭제 |
| 📅 2개월 후 삭제 | 이전 데이터 60일간 보관 후 삭제 |
| 🗓️ 삭제 예정 관리 | 삭제 예정 목록 확인, 즉시삭제로 앞당기기 가능 |

### 기계·설비
| 기능 | 설명 |
|------|------|
| ➕ 추가 | 체크박스로 기계 추가 (자동 이력 기록) |
| ➖ 제거 | 즉시/1개월/2개월 후 제거 선택 |

### 시설(환경)
| 기능 | 설명 |
|------|------|
| 변경 감지 | 대기/수질/폐기물 변경 시 자동 감지 |
| 제거 옵션 | 즉시/1개월/2개월 후 제거 선택 |

### 변경이력
- 모든 변경사항 자동 기록 (시간, 카테고리, 내용)
- 카테고리별 필터링
- TXT 파일 다운로드

## 작동 흐름
```
MSDS PDF 업로드 → CAS 자동 추출 → KOSHA API 최신 조회 → 맞춤 체크리스트
                                                         ↕
                                            변경점관리 (추가/변경/삭제 이력)
```

## 설치 & 실행
```bash
pip install streamlit pdfplumber requests
streamlit run app.py
```

## 7단계 위저드
①기본정보 → ②MSDS관리 → ③기계·설비 → ④시설현황 → ⑤진단결과 → ⑥대시보드 → ⑦변경이력

## ⚠️ 주의
- 화학물질 규제는 KOSHA API 최신 데이터만 사용 (내장DB 없음)
